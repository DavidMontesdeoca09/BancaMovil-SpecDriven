# Prompt IA - Generación del sistema Banca Móvil

Actúa como un desarrollador experto en C#.

Basándote en la siguiente especificación:

- Sistema de banca móvil
- Registro de usuario
- Login
- Consulta de saldo
- Transferencias
- Historial de transacciones

Genera un proyecto en C# tipo consola que incluya:

1. Clases principales (Usuario, Cuenta, Transaccion)
2. Métodos para cada funcionalidad
3. Menú interactivo en consola
4. Buenas prácticas de programación
5. Comentarios en el código

El código debe ser claro, modular y funcional.
