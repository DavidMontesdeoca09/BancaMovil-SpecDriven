# Especificación del Sistema - Banca Móvil

## 1. Descripción General
El sistema permitirá a los usuarios gestionar sus cuentas bancarias desde una aplicación móvil.

## 2. Objetivo
Facilitar operaciones bancarias de forma segura y rápida.

## 3. Funcionalidades principales
- Registro de usuario
- Inicio de sesión
- Consulta de saldo
- Transferencias entre cuentas
- Historial de transacciones

## 4. Usuarios del sistema
- Clientes del banco

## 5. Requisitos no funcionales
- Seguridad en autenticación
- Disponibilidad 24/7
- Interfaz fácil de usar
